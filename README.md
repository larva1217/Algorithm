# 🧠알고리즘

---

## 📚 수업 개요
본 과목은 컴퓨터를 이용한 문제 해결의 근간이 되는 자료 구조와 알고리즘을 학습하는 것을 목표로 한다.

트리, 그래프, 정렬 알고리즘, 이진 탐색 트리, 균형 탐색 트리 등 다양한 자료 구조와 알고리즘을 이해하고 구현함으로써 효율적인 프로그램을 설계하고 개발할 수 있는 능력을 기른다.

---

## 🎯 교과 목표
- 다양한 자료 구조와 알고리즘을 이해한다.
- 트리, 그래프, 정렬, 탐색 구조를 익힌다.
- 이를 활용하여 복잡한 문제를 해결할 수 있는 능력을 기른다.
- 효율적인 프로그램 설계 및 구현 능력을 향상시킨다.

---

## 📌 주요 학습 내용
- 자료 구조의 기본 개념
- 배열, 연결 리스트
- 스택, 큐
- 트리 구조 (이진 트리, BST, AVL 등)
- 그래프 구조 및 탐색 (DFS, BFS)
- 정렬 알고리즘 (버블, 선택, 삽입, 퀵, 병합 정렬 등)
- 탐색 알고리즘 (이진 탐색 등)
- 균형 탐색 트리

---

## 💻 선수 학습
- 자료구조 기초

---

## 🧑‍🏫 수업 방식
- 강의식 (60%)
- 실습/실험 (40%)
